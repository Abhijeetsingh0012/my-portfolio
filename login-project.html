Login & Registration System

PHP & MySQL Based Authentication Project

Project Overview

This project is a secure user authentication system developed using PHP and MySQL. It allows users to register, login, and access a protected dashboard. We can change profile name and also change profile image. This Login system use in admin pannel in future upgrade.

Features

  • User Registration
  • User Login
  • Form Validation
  • Password Security
  • Session Management
  • Logout System
  • Change Photo
  • Update Profile

Technologies Used

HTML CSS JavaScript PHP MySQL
